# 力学 期末試験 完全攻略ガイド（2023 & 2025）

力学の期末試験を、**授業に一度も出ていない人でも理解できる**ように解説する学習サイトです。
**2023年度（8月4日）と2025年度（8月4日）の全56問**を収録し、年度で絞り込めます。

## 特徴

各問題を4ステップで解説：

1. **📄 問題文** — 試験に出た問題をそのまま掲載
2. **🤔 問題の意味** — 「そもそも何を聞かれているのか」を日本語に翻訳。必要な用語・公式もここで解説
3. **💡 ヒント** — 段階的に開けるヒント。いきなり答えを見ずに自力で挑戦できる
4. **✅ 解説** — 途中式を省略しないステップ形式の完全解説 + 検算

その他の機能：

- 🗓 **年度フィルタ**（すべて / 2023年度 / 2025年度）
- 📚 つまずきやすい用語のミニ辞典（16項目）
- ✅ クリア済み問題の進捗管理（ブラウザに保存）
- 🌙 ダークモード対応
- 📱 スマホ対応
